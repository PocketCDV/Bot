Basically, there are 2 kinds of notifications - daily and upcoming. daily are notifications which are sent every morning,
and tell about all classes this day. upcoming are send an hour before the specific class. daily notifications have an interesting
timing feature - timestamp function is a min(9.00 AM , start time of the earliest class - 1 HOUR). so, for example, if first class
is at 8:30 AM - notification is at 7:30, but if class is at 11:15 - notification is at 9 AM.
if there are no classes today - daily notification is not sent.
